Admin Login

Manage reviews and read contact submissions. Data lives in this browser’s localStorage (same site origin as the public pages).

Admin Panel

Reviews use lpm_reviews. Contact form uses lpm_messages. Submissions only appear here if they were sent from this same domain in this browser.

Reviews on site

Same list visitors see on the home page. New reviews publish automatically. Use Remove to delete one for everyone.

Contact — inbox

Use Compose in Gmail to open Gmail in this browser (e.g. Brave) so you send from the Google account you’re signed into there. Mail app uses Windows’ default mail handler. Submissions from contact.html.
